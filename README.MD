- Vitest e React Testing Library
- Styled Components
- React

### Get Started

```sh
git clone https://github.com/guilhermeabell/volunteer-challenge-frontend
cd volunteer-challenge-frontend
npm install
npm run dev
```

### Run Tests

```sh
npm run test
```
