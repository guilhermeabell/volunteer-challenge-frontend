### Cloning the repository

```shell
git clone https://github.com/guilhermeabell/volunteer-challenge-frontend
cd volunteer-challenge-frontend
```

### Install packages

```shell
npm install

```

### Start the app

```shell
npm run dev
```

### Run Tests

```sh
npm run test
```
